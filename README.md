# NgStorybook

## 概要

- Angular + Storybook のサンプルです。

## 実行環境

- Node.js - 12.x
- Yarn - 1.17.x

## 使用ライブラリ

- Angular - 9.1.x
- @storybook/angular - 5.3.x

## 動作確認

### 1. サンプルのダウンロード

```bash
git clone https://github.com/yasu-s/ng-storybook-sample.git
```

### 2. ブランチ切り替え

```bash
cd ng-storybook-sample
git checkout develop/minimum
```

### 3. パッケージインストール  

```bash
yarn
```

### 4. サンプルの起動  

```bash
yarn storybook
```
